# 🔥 Бот «Сожги это»

Бот для эмоциональной разгрузки: пользователь пишет — сообщение мгновенно исчезает.

---

## Шаг 1 — Создай бота в Telegram

1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Придумай имя (например: `Сожги это`)
4. Придумай username (например: `burn_it_bot`)
5. Скопируй токен — он выглядит так: `123456789:AAF...`

---

## Шаг 2 — Загрузи на Railway

1. Зайди на [railway.app](https://railway.app) и зарегистрируйся через GitHub
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи эти 3 файла в репозиторий GitHub:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
4. Railway сам найдёт Procfile и запустит бота

---

## Шаг 3 — Добавь токен

В Railway открой свой проект → вкладка **Variables** → добавь:

```
BOT_TOKEN = вставь_свой_токен_сюда
```

---

## Шаг 4 — Готово!

Бот запустится автоматически. Найди его в Telegram по username и напиши что-нибудь — сообщение исчезнет через секунду.

---

## Важно про удаление сообщений в группе/канале

Чтобы бот мог удалять сообщения участников в группе:
- Добавь бота в группу
- Назначь его **администратором** с правом «Удаление сообщений»
